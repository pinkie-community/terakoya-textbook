# AI寺子屋 フィードバック収集システム

## 🎯 システムの目的

AI寺子屋の**継続的改善**のため、参加者・ファシリテーター・コミュニティメンバーからの意見を体系的に収集し、教材・ワークショップ・コミュニティ運営に反映させるシステムです。

### **フィードバック品質向上**

#### **回答率向上施策**
```
- タイミング最適化: ワークショップ直後の温度感が高い時期
- 簡便性確保: 5分以内で回答可能な設計
- 価値提示: 「皆さんの意見で教材が改善されます」
- 感謝表現: 回答者への感謝とフィードバック実装報告
- 追跡配慮: しつこい催促は避け、任意性を尊重
```

#### **回答品質向上施策**  
